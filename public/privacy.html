นโยบายความเป็นส่วนตัว · Privacy Policy

มีผลตั้งแต่: 26 พฤษภาคม 2026 · ผู้ดูแลข้อมูล: PostPost · ติดต่อ: support@postpost.app

PostPost ("เรา") ให้ความสำคัญกับความเป็นส่วนตัวของคุณ นโยบายนี้อธิบายว่าเราเก็บข้อมูลอะไร ใช้ทำอะไร และคุณมีสิทธิอะไรบ้างเกี่ยวกับข้อมูลของคุณ

1. ข้อมูลที่เราเก็บ

1.1 ข้อมูลบัญชี (Account data)

  • อีเมล + ชื่อ ที่คุณใช้สมัครหรือ login
  • รหัสผ่าน (เก็บแบบ hash — เราไม่เห็น plaintext)
  • workspace / brand profile ที่คุณตั้งขึ้น (ชื่อ, คำอธิบาย, voice, archetype)

1.2 ข้อมูลจากบัญชีโซเชียลที่คุณเชื่อม

  • Facebook Page: Page ID, Page Access Token, ชื่อเพจ, จำนวน Followers (อ่านจาก Graph API หลังคุณกด Connect)
  • Instagram Business: Instagram Business Account ID, username (อ่านจาก Graph API ผ่าน FB Page ที่ผูกอยู่)
  • TikTok: Open ID, Access Token, Refresh Token (อ่านจาก TikTok API หลังคุณกด Connect)

เราใช้ token พวกนี้เพื่อโพสต์เนื้อหาที่คุณสร้างขึ้นเมื่อคุณกดเผยแพร่เท่านั้น — เราไม่อ่านข้อความส่วนตัว ไม่ดูเพื่อน ไม่ดูคอมเมนต์เพื่อขายข้อมูล

1.3 ข้อมูลการใช้งาน (Usage data)

  • เหตุการณ์ในแอป (page view, ปุ่มที่กด) เพื่อปรับปรุง UI
  • Error log (เมื่อเกิด crash) เพื่อแก้บั๊ก
  • เก็บไว้ 90 วัน แล้วลบอัตโนมัติ

1.4 เนื้อหาที่คุณสร้าง (Content data)

  • caption / hook / hashtag / รูป / วิดีโอ ที่ AI ช่วยสร้างให้คุณ
  • สินค้าที่ดึงจาก Shopee/Lazada/TikTok Shop
  • เก็บไว้ใน workspace ของคุณเอง — workspace อื่นมองไม่เห็น (multi-tenant isolation)

2. เราใช้ข้อมูลทำอะไร

  • โพสต์อัตโนมัติ — เผยแพร่เนื้อหาที่คุณสั่งไปยังเพจ FB/IG/TikTok ที่คุณเชื่อม
  • AI Content Generation — ส่ง brand profile + topic เข้า AI model (Claude, Gemini, fal.ai) เพื่อสร้างข้อความ/รูป/วิดีโอ
  • Analytics — แสดงสถิติการโพสต์ของคุณ (engagement, reach) ดึงจาก Facebook Graph API
  • การปรับปรุงผลิตภัณฑ์ — วิเคราะห์ usage pattern (ไม่ระบุตัวบุคคล) เพื่อหา feature ที่ควรพัฒนา

3. เราแชร์ข้อมูลกับใคร

เราไม่ขายข้อมูลคุณให้ใคร แต่จำเป็นต้องส่งบางข้อมูลให้ third party เพื่อให้บริการทำงานได้:

  • Meta (Facebook, Instagram) — token + content เพื่อโพสต์ตามที่คุณสั่ง
  • TikTok — token + video เพื่อโพสต์
  • OpenAI / Anthropic / Google (Gemini) / fal.ai — text/image prompts เพื่อ AI generation
  • Supabase — database hosting (EU region)
  • Vercel — application hosting

4. การเก็บรักษาและความปลอดภัย

  • Token ของ FB/IG/TT เก็บใน database ที่ encrypt at-rest (Supabase)
  • การเชื่อมต่อทั้งหมดผ่าน HTTPS
  • รหัสผ่าน hash ด้วย bcrypt
  • เข้าถึง database ผ่าน service role key เท่านั้น — staff ของเราไม่เห็น raw token

5. สิทธิของคุณ

  • ดูข้อมูล — ขอ export ข้อมูล workspace ทั้งหมดได้ผ่าน support@postpost.app
  • แก้ไข — แก้ใน UI ของแอปได้เอง หรือขอผ่าน support
  • ลบ — ลบ workspace + ทุกข้อมูลที่เกี่ยวข้องได้ผ่าน Settings → Delete Account
    • หรือใช้ Facebook Data Deletion: /api/facebook/data-deletion
  • ยกเลิกการเชื่อมต่อ FB/IG/TT — กดปุ่ม "ยกเลิก" ที่ Profile page Token จะถูกลบทันที

6. Cookies

เราใช้ cookies เฉพาะที่จำเป็น:

  • postpost_token — JWT ของ session ของคุณ (httpOnly: false เพราะ frontend ต้องอ่านได้)
  • pp_session_id — ID ของ session สำหรับ analytics ไม่ระบุตัวบุคคล

เราไม่ใช้ third-party tracking cookies (Google Analytics, Facebook Pixel ฯลฯ)

7. ผู้ใช้อายุต่ำกว่า 13

PostPost ไม่ให้บริการกับเด็กอายุต่ำกว่า 13 ปี ถ้าเราตรวจพบว่ามี user ที่อายุน้อยกว่านี้ บัญชีจะถูกลบทันที

8. การเปลี่ยนแปลงนโยบาย

ถ้านโยบายนี้เปลี่ยนแปลงสำคัญ เราจะแจ้งคุณผ่านอีเมล + แสดง notice ในแอป ก่อนมีผล 30 วัน

9. ติดต่อเรา

คำถาม / ขอใช้สิทธิ / แจ้งปัญหา: support@postpost.app
